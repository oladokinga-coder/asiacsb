# Кнопка в таблице «Отправить письма»

При нажатии кнопки в Google Таблице вызывается ваш API — обрабатываются строки с **send = ok** и отправляются письма.

## Шаг 1: Откройте скрипт таблицы

1. Откройте вашу Google Таблицу (лист «Транзакции»).
2. Меню **Расширения** → **Apps Script**.
3. Удалите весь код в редакторе и вставьте код из файла `Script.gs` (см. ниже).
4. В коде замените:
   - `ВАШ_URL_САЙТА` — например `https://ваш-проект.vercel.app` (без слэша в конце).
   - `ВАШ_CRON_СЕКРЕТ` — если в Vercel задали `CRON_SECRET`, вставьте его сюда; если нет — оставьте пустую строку `''`.

## Шаг 2: Сохраните и разрешите запуск

1. Нажмите **Сохранить** (дискета), задайте имя проекта (например «Отправка писем»).
2. Выберите функцию **sendTransactionEmails** в выпадающем списке и нажмите **Выполнить**.
3. При первом запуске Google попросит разрешения: **Проверить разрешения** → выберите ваш аккаунт → **Дополнительные** → перейти на страницу «…» → **Разрешить**.

## Шаг 3: Кнопка в таблице

1. В таблице: **Вставка** → **Рисунок** → нарисуйте или загрузите кнопку (или напишите текст в ячейке).
2. После вставки нажмите на рисунок → три точки справа вверху → **Назначить скрипт**.
3. Введите имя функции: **sendTransactionEmails** → OK.

Готово. При нажатии на кнопку скрипт вызовет ваш сайт и отправит письма по строкам, где в колонке **send** указано **ok**.
