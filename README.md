# Česká Banka — сайт банка с личным кабинетом и Google Таблицей

Масштабный проект банковского сайта в стиле ČSOB: лендинг, регистрация/вход, личный кабинет (баланс, карта, история операций, профиль). Данные клиентов и транзакции отображаются в **Google Таблице**; вы можете вручную вносить зачисления/списания в таблицу — они сразу отображаются у клиента.

## Возможности

- **Главная страница** — лендинг с навигацией (Услуги, О банке, Вход, Открыть счёт).
- **Регистрация и вход** — реальная регистрация с сохранением в БД и добавлением строки в Google Таблицу.
- **Личный кабинет:**
  - Обзор: баланс, карта, последние операции.
  - Карта: данные банковской карты.
  - История: все транзакции по счёту.
  - Профиль: имя, email, баланс, дата открытия счёта.
- **Google Таблица:**
  - Лист **Клиенты**: все зарегистрированные клиенты (userId, email, fullName, balance, cardNumber, cardValid, createdAt). Вы можете править баланс (balance) вручную.
  - Лист **Транзакции**: добавляете строки (email клиента, amount, type, description, date) — они отображаются у клиента в кабинете.

## Технологии

- **Next.js 14** (App Router), React, TypeScript
- **Prisma + SQLite** — пользователи и сессии
- **Google Sheets API** (сервисный аккаунт) — чтение/запись таблицы
- **bcrypt** — хеширование паролей

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Создайте файл `.env` в корне (скопируйте из `.env.example`):
   ```env
   DATABASE_URL="file:./dev.db"
   ```
   Для работы с Google Таблицей добавьте переменные — см. раздел ниже.

3. Инициализируйте БД:
   ```bash
   npm run db:generate
   npm run db:push
   ```
   (или `npx prisma generate` и `npx prisma db push`)

4. Запустите сервер:
   ```bash
   npm run dev
   ```

   Сайт будет доступен по адресу [http://localhost:3000](http://localhost:3000).

## Настройка Google Таблицы

Подробная инструкция: **[docs/GOOGLE_SHEET_TEMPLATE.md](docs/GOOGLE_SHEET_TEMPLATE.md)**.

Кратко:

1. Создайте Google Таблицу с двумя листами: **Клиенты** и **Транзакции**.
2. В «Клиенты» первая строка: `userId`, `email`, `fullName`, `balance`, `cardNumber`, `cardValid`, `createdAt`.
3. В «Транзакции» первая строка: `id`, `email`, `amount`, `type`, `description`, `date`.
4. В Google Cloud создайте сервисный аккаунт, включите Google Sheets API, скачайте JSON-ключ.
5. Делитесь таблицей с email сервисного аккаунта (права «Редактор»).
6. В `.env` укажите:
   - `GOOGLE_SHEET_ID` — ID таблицы из URL.
   - `GOOGLE_SERVICE_ACCOUNT_EMAIL` — из JSON.
   - `GOOGLE_PRIVATE_KEY` — поле `private_key` из JSON (можно с `\n` в одной строке).

Без этих переменных сайт работает: регистрация и вход сохраняются в БД, но баланс/карта/транзакции в кабинете будут пустыми до подключения таблицы.

## Как вносить транзакции и баланс

- **Зачисление:** в листе «Транзакции» добавьте строку: email клиента, положительная сумма (например 5000), type «Зачисление», description «Зарплата», date (например 08.03.2025). В листе «Клиенты» обновите ячейку **balance** у этого клиента (увеличьте на сумму зачисления).
- **Списание:** то же, но amount отрицательный (например -500), и уменьшите balance в «Клиенты».

Клиент видит обновлённые данные при следующей загрузке страницы кабинета (данные читаются из таблицы при каждом запросе).
